# liu-skills

个人编写的 [Cursor Agent Skills](https://cursor.com/docs/agent/skills) 合集。

## Skills

| Skill | 说明 |
|-------|------|
| [apple-reminders](./apple-reminders/) | 在 macOS 上通过 AppleScript 管理 Apple 提醒事项（查看、创建、更新） |

## 安装

将需要的 skill 目录复制到 Cursor skills 目录：

```bash
cp -r apple-reminders ~/.cursor/skills/
```

首次使用 `apple-reminders` 时，请在 **系统设置 → 隐私与安全性 → 自动化** 中，为 Terminal / Cursor 授予 **提醒事项** 权限。

## 许可

各 skill 默认可自由使用与修改。
